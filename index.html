Home

A small piece of code that changed the whole internet

This site is about the HTTP cookie, that tiny bit of text a website stores in your browser so it can remember you the next time you visit. It sounds small, but this one idea from 1994 ended up shaping online shopping, login systems, advertising, and a good chunk of privacy law around the world. This site walks through where the cookie came from, why it became such a big privacy argument, and where the cookie seems to be headed next as browsers try to phase parts of it out.

This project started as a research essay for IT 3203 and is now growing into a full website. Use the menu above, or the links below, to move between pages.

Key Terms

HTTP Cookie

A short piece of text a website asks your browser to store and send back later.

First Party Cookie

A cookie set by the actual site you are visiting, used for logins and carts.

Third Party Cookie

A cookie set by an outside script, usually an ad company, on many sites at once.

Cookie Consent

The popup banner that asks permission before a site sets non essential cookies.

See the full key concepts page for more terms and longer explanations.
