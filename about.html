About This Project

This site started as a research essay on the history of the HTTP cookie for Assignment 1 in this course and it's now turning into a full website for the class project. Later milestones will add an interactive quiz and make the site fully responsive for smartphones.

This is a class project for IT 3203, Web Development, at Kennesaw State University. Visit the class website at ksuweb.github.io/IT3203.
